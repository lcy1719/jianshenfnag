### 万字文档+健身房管理系统+SSM+Vue
## 项目采用技术：
开发环境：IDEA、JDK1.8、Maven、MySQL、VSCode
技术栈：Java、MySQL、Spring、SpringMVC、Mybatis/MybatisPlus、Vue、JavaScript、CSS

系统功能模块：
本系统包含管理员、会员2个角色，
管理员：账号管理、健身房信息管理、教练信息管理、健身器材列表管理、课程列表管理、添加类别、登录、信息添加、信息删除、信息修改等功能。
会员：会员信息管理、会员卡信息管理、个人信息查询、个人信息修改、健身区域查看、课程预约等功能。
